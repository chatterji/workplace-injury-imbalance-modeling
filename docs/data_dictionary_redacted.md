# Redacted Data Dictionary

The original dataset is confidential and is not included in this repository.

## Target
- `CLAIM_AGG`: Binary target indicating whether a workplace injury claim occurred.

## Example public/demo feature categories
Use high-level descriptions only in the public repo. Replace or expand these with redacted business-safe descriptions:
- shift workload metrics
- tenure-related metrics
- overtime or schedule-related variables
- prior absence or incident history variables
- operational activity measures

## Public guidance
Do not publish:
- employee identifiers
- company-specific codes
- exact internal field definitions that would expose proprietary processes
- raw claim-level data
